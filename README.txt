업무 분석 도구 (계정과목 증감 분석 + PC 실사대장 + 실사 결과 조회)
====================================================================

실행 방법
----------
1. index.html 파일을 브라우저에서 엽니다. (홈 화면)
   - 상단 네비(링크): "홈" | "계정과목 증감 분석" | "PC 실사·실사결과"
   - 카드 클릭 시 해당 페이지로 이동: 계정과목 → account/, PC 실사·실사결과 → asset/
   - CDN: SheetJS(xlsx) 사용. 계정과목은 CSV 또는 엑셀(.xlsx/.xls) 지원.

2. 계정과목 증감 분석 (account/index.html)
   - CSV 또는 엑셀(.xlsx/.xls) 파일을 클릭 또는 드래그하여 업로드. (CSV는 UTF-8, 헤더 필수)
   - 필수 컬럼: 기표일자, 계정과목, 금액(또는 계정과목에 대한 금액), 적요
   - 최소 2개월 이상 데이터 필요. 업로드 후 자동 분석. 처리 중 "파일 로딩 중..." 표시.

3. PC 실사대장 (asset/index.html — "PC 실사대장" 서브탭)
   - 엑셀(.xlsx/.xls) 파일을 클릭 또는 드래그하여 업로드.
   - 헤더에 부서, 사용자명, IP, 제조연월, 사용년수, 운영체제 포함. "구분"(DT/NT 등) 컬럼 있으면 표시.
   - 업로드 후 테이블·요약 카드 표시, 행 클릭 시 교체 체크리스트 모달에서 17개 항목 체크. "처리 중…" 로딩 표시.

4. 실사 결과 조회 (asset/index.html — "실사 결과 조회" 서브탭)
   - "엑셀 내보내기"로 저장한 PC실사결과(.xlsx) 파일을 클릭 또는 드래그하여 업로드.
   - 전체/교체권고/모니터링/정상 요약 카드, 부서·판정 필터, 읽기 전용 테이블. "다른 파일 분석하기"로 재업로드 가능.


■ 1. 계정과목 증감 분석
-----------------------
- 업로드: 클릭 또는 드래그 앤 드롭. (CSV, .xlsx, .xls)
- 상세 보기: 월 쌍별 계정과목 증감, 적요별 차이 내역.
- 요약 보고서: 계정과목별 요약, Top3 적요·기여도·적요건수, 기여도 불일치 시 경고 배지.
- 필터: 증가/감소, 최소 |차이| (debounce 적용).
- 정렬: 요약 테이블 헤더 클릭 (계정과목, 전월/차월금액, 차이, 적요건수).
- 요약 CSV 다운로드: 현재 필터 적용 결과만 UTF-8 BOM으로 저장.

데이터 규칙
  - 금액: "계정과목에 대한 금액" 우선, 없으면 "금액".
  - 적요: 월 표현 제거 후 그룹화. 차이=차월−전월, 0인 항목 미표시.
  - 디버그: account/index.html?debug 로 기여도 불일치 콘솔 출력.


■ 2. PC 실사대장 (노트북 교체 체크리스트)
-----------------------------------------
- 엑셀 업로드 후 장비 목록 표시. 부서/OS/사용년수/상태 필터, 검색 지원.
- 구분: 엑셀 "구분" 컬럼(DT, NT 등)을 사용년수 뒤 열·모달·엑셀 내보내기에 표시.
- 교체점수: 17개 체크 항목 각 1~3점, 선택한 항목 점수 합산. 7점 이상 교체권고, 5~7점 모니터링, 4점 이하 정상.
- 자동 체크: 사용년수 5년 이상 → "사용기간 5년 이상"(3점), OS에 "10" 포함 → "OS 지원 종료 예정 Win10"(3점). 모달에서 수정 가능.
- 모달: 행 클릭 시 17개 체크 항목 표시. "저장 후 닫기"로 점수 반영. 저장 없이 닫을 때 확인 메시지.
- 엑셀 내보내기: 전체 실사결과(체크 항목 ● 포함), 교체권고 대상, 요약 시트가 포함된 xlsx 다운로드.
- 인쇄:
  - 체크 0개 또는 2개 이상: 전체 리스트(테이블) 인쇄.
  - 1개만 선택 후 인쇄: 해당 1건만 "노트북 교체대상 여부 체크리스트" 형식으로 인쇄.
- 필터 결과 0건 시 테이블에 "조건에 맞는 데이터가 없습니다." 표시.


■ 3. 실사 결과 조회
--------------------
- 용도: PC 실사대장에서 "엑셀 내보내기"로 저장한 파일을 다시 올려 통계·목록만 조회. (편집 불가)
- 업로드: 내보낸 형식(.xlsx)의 "전체 실사결과" 시트 또는 첫 시트 사용.
- 요약 카드: 전체 장비, 교체권고, 모니터링, 정상 대수.
- 필터: 부서, 판정(교체권고/모니터링/정상). 필터 결과 0건 시 "조건에 맞는 데이터가 없습니다." 표시.
- 다른 파일 분석하기: 툴바 버튼으로 업로드 화면 복귀 후 새 파일 선택 가능.


파일 구성
----------
- index.html            : 홈 전용. 상단 네비 + 카드(계정과목/PC실사/추가예정). 카드 클릭 시 해당 페이지로 이동.
- css/common.css        : 공통 스타일 (.app-nav, .nav-links, .nav-link 등).
- js/common.js          : 공통 유틸 (escapeHtml).
- account/index.html    : 계정과목 증감 분석 단일 페이지.
- asset/index.html      : PC 실사대장 + 실사 결과 조회 (서브탭 2개) 단일 페이지.
- asset-checklist-points.md : PC 실사대장 17개 항목·점수·판정 기준 문서.


기술 사양
----------
- 폰트: Noto Sans KR, JetBrains Mono (Google Fonts).
- 계정과목: 순수 JavaScript, CSV 자동 구분자 감지(쉼표/세미콜론/탭), 드래그 앤 드롭 업로드.
- PC 실사대장·실사 결과 조회: SheetJS(xlsx) CDN. 엑셀 읽기·내보내기, 드래그 앤 드롭, 파싱 중 로딩 표시.
- 반응형: 768px 이하에서 탭 줄바꿈, 요약 카드 2열.
- 접근성: 탭 role/aria-selected/aria-hidden, 포커스 링크 스타일, 모달 role="dialog"·aria-modal·aria-labelledby.
- 브라우저: Chrome, Edge, Firefox 등 최신 브라우저.


주의사항
----------
- 계정과목: CSV UTF-8, 기표일자·계정과목·금액·적요 필수, 최소 2개월 데이터.
- PC 실사대장: 엑셀 헤더에 부서·사용자명 등 필수. 체크 후 "저장 후 닫기" → "엑셀 내보내기"로 저장 권장.
- 실사 결과 조회: 엑셀 내보내기로 저장한 파일만 업로드. 다른 형식이면 오류 가능.
